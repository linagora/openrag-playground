# Souveraineté des données

**OpenRAG Playground** est conçu avec la souveraineté numérique européenne au coeur de son architecture.

**Auto-hébergé.** L'application complète fonctionne sur votre infrastructure. Aucune dépendance cloud, aucun enfermement SaaS.

**Zéro requête externe.** L'application n'effectue aucun appel vers un service externe. Toutes les ressources sont hébergées localement. Votre navigateur ne communique qu'avec votre serveur.

**Open source.** Le code source complet est disponible sous licence AGPL-3.0, garantissant transparence, auditabilité et indépendance technologique.

**Fournisseur européen.** Développé par [LINAGORA](https://linagora.com), basé à Toulouse, France. Soumis au droit européen et au RGPD.

**Vos données restent les vôtres.** Documents, tokens et configuration ne quittent jamais votre infrastructure. Pas de télémétrie, pas de partage de données, pas de traitement par des tiers.
