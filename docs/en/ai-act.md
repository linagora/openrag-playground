# AI Transparency (AI Act)

In accordance with the European AI Act (Regulation EU 2024/1689), we provide the following transparency information:

**AI system:** This application is a user interface to [Open-RAG](https://open-rag.ai), a Retrieval-Augmented Generation (RAG) system.

**Purpose:** Answer user questions by retrieving relevant documents from a knowledge base and generating natural language responses.

**How it works:** When you ask a question, the system searches indexed documents for relevant passages, then uses a large language model to generate an answer grounded in those sources. Cited sources are displayed alongside answers.

**Risk classification:** This system is not classified as high-risk under the AI Act. It serves as a productivity and information retrieval tool.

**Human oversight:** All responses should be verified by the user. The system may produce inaccurate or incomplete answers. Sources are provided for verification.

**Provider:** LINAGORA -- [linagora.com](https://linagora.com)
