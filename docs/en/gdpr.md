# Data Protection (GDPR)

**Legal basis.** The processing of personal data is based on the performance of a contract (Article 6(1)(b) GDPR). Your data is processed as necessary to provide you with the service you are using.

**Data processed.** Documents uploaded to the platform, queries submitted to the AI system, and associated metadata (timestamps, partition names). API access tokens are stored encrypted on the server.

**Zero external requests.** This application makes no requests to any third-party service. All assets (fonts, scripts, styles) are self-hosted. Your browser communicates only with the server hosting this application.

**No tracking.** No analytics, no fingerprinting, no advertising. The only cookie used is a signed, httpOnly session cookie for authentication, expiring after 24 hours.

**Data retention.** Data is retained for the duration of the contract. Uploaded documents are stored on the OpenRAG server infrastructure. Configuration data is stored locally on the application server.

**Your rights.** You have the right to access, rectify, erase, restrict, and port your personal data, as well as to object to processing. Contact the DPO to exercise your rights.
