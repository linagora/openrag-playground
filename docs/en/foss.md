# Free and Open Source Software

Free and Open Source Software (FOSS) is software that respects users' freedom and community. It guarantees that anyone can use, study, modify, and distribute the software without restriction. FOSS is not merely a development model -- it is a commitment to transparency, collaboration, and digital autonomy.

## The Four Freedoms

The Free Software Foundation defines four essential freedoms that software must grant to qualify as free software:

- **Freedom 0 -- Run.** The freedom to run the program as you wish, for any purpose.
- **Freedom 1 -- Study and modify.** The freedom to study how the program works and change it so it does your computing as you wish. Access to the source code is a precondition for this.
- **Freedom 2 -- Redistribute.** The freedom to redistribute copies so you can help others.
- **Freedom 3 -- Distribute modified versions.** The freedom to distribute copies of your modified versions to others. This gives the whole community a chance to benefit from your changes. Access to the source code is a precondition for this.

## Benefits of FOSS

### Transparency

Every line of code is open for inspection. There are no hidden behaviors, no concealed data collection, and no undisclosed backdoors. You can verify exactly what the software does.

### Security Through Auditing

Open source code can be reviewed by anyone -- independent researchers, government agencies, or your own security team. Vulnerabilities are found and fixed faster when thousands of eyes can examine the code.

### No Vendor Lock-in

FOSS prevents dependency on a single vendor's ecosystem. You retain full control over your infrastructure and data. If a provider disappears or changes direction, the software and your investment remain yours.

### Digital Sovereignty

For organizations and public institutions, FOSS is a cornerstone of digital sovereignty. It ensures that critical systems are not controlled by foreign entities and that data governance stays where it belongs -- with the data owner.

### Community-Driven Innovation

FOSS benefits from contributions by a global community of developers, researchers, and organizations. This collaborative model accelerates innovation and produces software shaped by real-world needs rather than a single company's roadmap.

### Cost Efficiency

FOSS eliminates licensing fees and reduces total cost of ownership. Resources that would go toward proprietary licenses can be redirected to customization, support, and improvement of the software itself.
