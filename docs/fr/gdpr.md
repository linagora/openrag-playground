# Protection des données (RGPD)

**Base légale.** Le traitement des données personnelles repose sur l'exécution d'un contrat (Article 6(1)(b) du RGPD). Vos données sont traitées dans la mesure nécessaire à la fourniture du service que vous utilisez.

**Données traitées.** Documents téléversés sur la plateforme, requêtes soumises au système d'IA, et métadonnées associées (horodatages, noms de partitions). Les tokens d'accès API sont stockés chiffrés sur le serveur.

**Zéro requête externe.** Cette application n'effectue aucune requête vers un service tiers. Toutes les ressources (polices, scripts, styles) sont hébergées localement. Votre navigateur ne communique qu'avec le serveur hébergeant cette application.

**Aucun traçage.** Pas d'analytics, pas de fingerprinting, pas de publicité. Le seul cookie utilisé est un cookie de session signé et httpOnly pour l'authentification, expirant après 24 heures.

**Conservation des données.** Les données sont conservées pendant la durée du contrat. Les documents téléversés sont stockés sur l'infrastructure du serveur OpenRAG. Les données de configuration sont stockées localement sur le serveur applicatif.

**Vos droits.** Vous disposez des droits d'accès, de rectification, d'effacement, de limitation, de portabilité de vos données personnelles, ainsi que du droit d'opposition au traitement. Contactez le DPO pour exercer vos droits.
