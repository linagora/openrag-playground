# About OpenRAG Playground

**OpenRAG Playground** is a demonstration interface for [Open-RAG](https://open-rag.ai), the sovereign Retrieval-Augmented Generation platform.

Built for live demos and client presentations, it lets operators showcase Open-RAG's capabilities through realistic persona-based scenarios -- each backed by real API tokens and access controls.

Developed by [LINAGORA](https://linagora.com), a European open-source software company dedicated to digital sovereignty.
